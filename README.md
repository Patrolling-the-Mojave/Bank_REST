# Bank_Rest

Bank_Rest — это монолитное RESTful приложение для управления банковскими картами и пользователями с ролевой моделью (USER, ADMIN), JWT-аутентификацией и шифрованием конфиденциальных данных.

# Спецификация API
https://patrolling-the-mojave.github.io/Bank_REST/

## Архитектура

Система состоит из следующих модулей:

- **Модуль аутентификации** — регистрация, вход, обновление JWT токенов.
- **Модуль пользователей** — управление профилями (администратор и пользователь).
- **Модуль карт** — создание, блокировка, активация, переводы между картами.
- **Модуль транзакций** — просмотр истории переводов.
- **PostgreSQL** — единая база данных.
- **Spring Security + JWT** — аутентификация и авторизация.

Все финансовые операции атомарны и выполняются в рамках ACID-транзакций. Номера карт хранятся в зашифрованном виде и маскируются при выводе.

## Запуск
1. Склонируйте репозиторий
2. Соберите проект: mvn clean package -DskipTests
3. Запустите контейнеры: docker-compose up --build
