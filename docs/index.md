# Bank_Rest Documentation

Добро пожаловать в документацию проекта Bank_Rest — монолитного приложения для управления банковскими картами и пользователями с ролевой моделью (USER, ADMIN) и JWT-аутентификацией.

## 🚀 Быстрый старт
- [Архитектура системы](./architecture.md)
- [Пользовательские сценарии](./user-scenarios.md)
- [API Документация](http://localhost:8080/swagger-ui.html)

Проект реализует безопасное хранение данных (шифрование номеров карт), атомарные финансовые операции и разделение доступа на основе ролей. Все действия логируются, а конфиденциальная информация маскируется при выводе.
